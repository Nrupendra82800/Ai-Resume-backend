# 🚀 MakeMyResume Backend v2 — PostgreSQL + Prisma

A scalable and production-ready backend application built using **Node.js**, **Express.js**, **PostgreSQL**, and **Prisma ORM**.

This project powers a Resume Builder platform with secure APIs, database integration, and clean backend architecture.

---

# 📌 Features

- REST API Architecture
- PostgreSQL Database Integration
- Prisma ORM
- CRUD Operations
- Environment Variable Configuration
- Error Handling Middleware
- Scalable Folder Structure
- Production Ready Backend Setup
